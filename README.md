1) What is the difference between null and undefined?
ans: undefined বাই ডিফল্ট এসইন হয়,আর নাল ডেভল্পার এসাইন করে।নাল অবজেক্ট টাইপ।ভ্যালু ক্লিয়ার করার সময় নাল ব্যাবহার করা হয়।
2) What is the use of the map() function in JavaScript? How is it different from forEach()?
ans: map() ব্যবহার করা হয় একটি অ্যারের প্রতিটি উপাদান পরিবর্তন করে নতুন একটি অ্যারে তৈরি করার জন্য।forEach() ব্যবহার করা হয় অ্যারের প্রতিটি উপাদানের উপর কোনো কাজ চালানোর জন্য, কিন্তু এটি কোনো নতুন অ্যারে রিটার্ন করে না।
3) What is the difference between == and ===?
ans: মান তুলনা করে, কিন্তু ডাটা টাইপ তুলনা করে না।মান এবং টাইপ দুটোই তুলনা করে।
4) What is the significance of async/await in fetching API data?
ans: async ব্যবহার করা হয় একটি ফাংশনকে asynchronous বানানোর জন্য।একটি ফাংশনকে Promise return করার ক্ষমতা দেয়. await ব্যবহার করা হয় Promise এর রেজাল্ট পাওয়ার জন্য অপেক্ষা করতে।Promise resolve না হওয়া পর্যন্ত পরবর্তী লাইন চালায় না
5) Explain the concept of Scope in JavaScript (Global, Function, Block).
ans: কোন ভ্যারিয়েবল কোথা থেকে ব্যবহার করা যাবে।Function Scope যে ভ্যারিয়েবল ফাংশনের ভিতরে ডিক্লেয়ার করা হয়, সেটি শুধু ওই ফাংশনের ভিতরেই কাজ করে। Block Scope এর ভিতরে ডিক্লেয়ার করা ভ্যারিয়েবল শুধু ওই block এর মধ্যেই থাকে।
Global Scope যে ভ্যারিয়েবল ফাংশনের বাইরে ডিক্লেয়ার করা হয়, সেটি Global Scope এ থাকে।
